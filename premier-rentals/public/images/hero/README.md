# Hero Images

Place your hero background image here:

| Filename       | Used in    | Recommended size |
|----------------|------------|------------------|
| `hero-bg.jpg`  | Homepage hero full-screen background | 1920×1080px minimum, landscape |

> **Tip:** Use `.jpg` for photos, `.webp` for best performance.
> If a file is missing, the site automatically falls back to a placeholder image.
